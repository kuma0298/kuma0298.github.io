img
(img) This tag is used to add images to an HTML page.
a
(a) This tag is used to link different web pages together.
h1-h6
(h1-h6) These tags are used to create headings in a web page, with h1 being the largest and h6 the smallest.
p
(p) This tag is used to write paragraphs in HTML pages.
title
(title) This tag is used to set the title of a web page.
br
(br) This tag is used to insert a line break.
form
(form) This tag is used to create an HTML form for input fields and buttons.
strong
(strong) This tag is used to emphasize important text.
